## About

A single-page photo portfolio application built with React. Features a responsive gallery layout organized by photography categories, an about section, and a contact form — all deployed to GitHub Pages.

## Features

- **Category Navigation** — Browse photos organized by category
- **Photo Gallery** — Responsive image grid with modal previews
- **About Section** — Photographer bio and background
- **Contact Form** — Form validation with user feedback
- **Single-Page App** — Smooth navigation without page reloads
- **GitHub Pages** — Live deployment with gh-pages

## Tech Stack

| Category | Technology |
|----------|------------|
| Framework | React 17 |
| Testing | React Testing Library 11, Jest |
| Deployment | GitHub Pages (gh-pages 4) |
| Build | react-scripts 5 |

## Getting Started

```bash
# Clone the repository
git clone https://github.com/coleyrockin/photo-port.git
cd photo-port

# Install dependencies
npm install

# Start the development server
npm start

# Deploy to GitHub Pages
npm run deploy
```

**Live Demo:** [coleyrockin.github.io/photo-port](http://coleyrockin.github.io/photo-port)
